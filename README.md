## 🚀 Sobre este proyecto

Este repositorio contiene un ejemplo simple de **Hola Mundo** creado para practicar **programación en Java y trabajo colaborativo con GitHub**.

El objetivo es aprender a:

* Utilizar **Git y GitHub**
* Trabajar con **ramas**
* Colaborar en proyectos de **Programación Orientada a Objetos (POO)**

---

## 💻 Ejemplo de código

```java
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola Mundo!");
    }
}
```
